## Overview
This project presents a comprehensive Power BI dashboard analyzing ski resorts worldwide. It focuses on infrastructure, pricing, experience quality, and seasonal availability to help identify the best destinations.


## Key Features
 Infrastructure Analysis (Lifts & Capacity)  
 Snow Reliability & Seasonality  
 Family-Friendly & Skill-Level Segmentation  
 Value vs Price Analysis  
 Summer & Night Skiing Insights

## Key Insights
- Resorts with higher lift capacity handle crowds more efficiently  
- Artificial snow systems improve reliability  
- Some resorts operate year-round  
- Family-friendly resorts have more beginner slopes  
- Not all expensive resorts provide the best value

## Business Value
This dashboard helps stakeholders:
- Compare ski resorts globally  
- Identify high-value destinations  
- Understand customer segments  
- Make data-driven tourism decisions
